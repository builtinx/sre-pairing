# Pairing Exercise for SRE

## Introduction

This holds pseudo and incomplete code for [Terraform](https://www.terraform.io/downloads.html) and [Docker](https://docs.docker.com/get-docker/). The code is to be modified in a feature branch during an interview session.

## Expectations

We're looking to get to know you in the following areas:
- Programming or scripting abilities
- AWS knowledge
- Written communication
- Documentation
- CICD operational knowledge

## Exercise Scenario
The hypothetic goal is to deploy a containerized web server in AWS. You are not expected to achieve this goal within the interview session.
